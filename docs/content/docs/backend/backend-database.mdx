# 后端数据库结构

## users 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 当前固定为默认租户 |
| username | string(64) | 登录用户名 |
| password_hash | string(256) | bcrypt 密码哈希 |
| display_name | string(64) | 显示名称 |
| avatar_url | string(512) | 头像 |
| role | enum(super_admin/tenant_admin/user) | 角色 |
| status | enum(active/inactive) | 状态 |

## credit_accounts 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 租户 ID |
| user_id | uint | 用户 ID |
| balance | int | 当前积分余额 |
| total_earned | int | 累计收入 |
| total_spent | int | 累计支出 |

## credit_transactions 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| account_id | uint | 账户 ID |
| type | enum(earn/spend/refund/adjust) | 流水类型 |
| amount | int | 变动积分 |
| balance_before | int | 变动前余额 |
| balance_after | int | 变动后余额 |
| ref_type | string(50) | 业务来源，如 image/video/audio/recharge |
| ref_id | string(100) | 关联业务 ID |
| note | string(500) | 备注 |
| metadata | longtext | 流水扩展详情 JSON，如模型、接口、计费单位、操作人 |

## credit_pricing 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 租户 ID |
| model | string(100) | 模型名 |
| credits_per_unit | int | 按次计费的单次积分消耗；动态视频计费可为 0 |
| unit_type | enum(per_image/per_video/per_video_second/per_token) | 计费单位 |
| pricing_mode | enum(per_unit/video_dynamic) | 计费方式；`video_dynamic` 表示按视频秒数和分辨率动态计费 |
| pricing_rule | longtext | 动态计费规则 JSON，例如视频基础积分和各分辨率秒单价 |

## tenant_api_configs 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 当前固定为默认租户 |
| base_url | string(500) | 上游 API 地址 |
| api_key | text | 加密后的上游 API Key |
| models_json | longtext | 模型目录 JSON |
| model_routes | longtext | 模型视频接口路由 JSON |

## channels 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| name | string(100) | 渠道显示名称，带索引 |
| base_url | string(500) | 上游 API 基础地址 |
| api_key | string(500) | 加密后的上游 API Key；模型 JSON 输出中隐藏该字段 |
| enabled | bool | 渠道是否启用，默认 true，带索引 |
| new_api_channel_id | int | 可为空，对应 new-api 的 channel_id |
| new_api_channel_id_val | string(100) | new-api 渠道人类可读映射名称 |
| metrics_base_url | VARCHAR(512) | NULL | 指标服务地址（为空则使用 base_url + /api） |
| remark | VARCHAR(500) | NULL | 管理员备注 |
| sync_status | string(20) | 模型同步状态：pending/syncing/success/failed |
| sync_error | string(500) | 最近一次同步错误信息 |
| synced_at | datetime | 可为空，最近一次成功同步时间 |

API Key 仅用于后端代理调用，按 `API_KEY_ENCRYPTION_KEY` 配置加密存储，不应出现在 JSON 响应中。

## channel_models 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| channel_id | uint | 所属渠道 ID，非空，带索引 |
| model_name | string(200) | 上游原始模型名，非空 |
| capabilities | string(100) | 模型能力 JSON 数组，可包含 image/video/text/audio |
| enabled | bool | 渠道内模型是否启用，默认 true |
| image_generate_route | string(30) | 图片生成路由：auto/generations/edits/chat/banana |
| image_edit_route | string(30) | 图片编辑路由 |
| video_route | string(30) | 视频路由：auto/openai/veo_json/waninter/yijia/xai/newapi/seedance |
| video_durations | string(200) | 视频时长 JSON 数组，例如 `[5,10]` |
| video_customizable | bool | 视频参数是否支持用户自定义，默认 false |
| sort_order | int | 前台展示排序，默认 0 |

`channel_id` 与 `model_name` 组成唯一约束 `idx_channel_model`，避免同一渠道重复记录同名模型。

## metrics_configs 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| metrics_base_url | string(500) | SuperAdmin 配置的 new-api 管理端基础地址；后端拼接 `/api/perf-metrics/channels?hours=N` 读取指标 |

该地址仅由后端请求，不保存或发送渠道 API Key，也不向普通用户暴露配置值。

## canvas_projects 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 租户 ID |
| user_id | uint | 所属用户 |
| project_id | string(128) | 前端画布 ID |
| title | string(200) | 画布标题 |
| data_json | longtext | 画布完整内容 |
| snapshot_json | longtext | 列表页摘要 |

## generation_records 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 租户 ID |
| user_id | uint | 所属用户 |
| record_id | string(128) | 前端记录 ID |
| type | enum(image/video/audio) | 记录类型 |
| title | string(200) | 标题 |
| payload_json | longtext | 生成记录完整内容 |

## model_call_logs 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 当前固定为默认租户 |
| user_id | uint | 调用用户 ID |
| username | string(64) | 用户名快照 |
| display_name | string(64) | 显示名称快照 |
| generation | string(20) | 生成类型 image/video/audio/text |
| model | string(100) | 调用模型 |
| channel_id | uint | 所属渠道 ID，非空，带索引 |
| channel_model_id | uint | 所属渠道模型 ID，非空，带索引 |
| method | string(10) | HTTP 方法 |
| path | string(255) | 上游接口路径 |
| status_code | int | 上游状态码，0 表示本地校验或网络错误 |
| error_message | string(500) | 错误摘要 |
| error_body | longtext | 上游错误响应体 |

管理后台模型健康概览基于该表统计近 24 小时、近 7 天失败量和 Top 失败模型。

## recharge_orders 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uint | 主键 |
| tenant_id | uint | 租户 ID |
| user_id | uint | 用户 ID |
| order_no | string(64) | 订单号 |
| credits | int | 充值积分 |
| amount | decimal | 支付金额 |
| status | enum(pending/paid/completed/failed) | 订单状态 |
| note | string(255) | 套餐备注 |
